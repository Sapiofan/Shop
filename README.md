# Shop-online

Any user can search, filter products, go to the description of a product, buy it, and ask questions to the shop. If a user is authenticated, additionally he can control cart and favorite products dynamically, and give reviews for any product. Adaptive design is present. An admin panel also exists, where an admin can add/change/remove products, control usersâ€™ feedback and change the content of the site.
